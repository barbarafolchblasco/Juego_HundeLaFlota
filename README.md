# HundeLaFlota
Código fuente y ejecutable del clásico juego "Hundir La Flota" programado en Java. 
<br><br>
El objetivo de este proyecto es implementar una versión básica del clásico juego de mesa Hundir
la Flota (también conocido como Batalla Naval) en una aplicación de consola en Java. Se ha
desarrollado la lógica del juego, la interacción con el usuario, la gestión de los tableros y las reglas
de colocación y disparo.
